  本项目的目的是 搭建一个基于spring boot的拿来即用的生产级别Java web框架

目前已整合：
-    spring boot
- 	 JPA
- 	 自定义数据源配置
- 	 基于AOP的全局事务管理
- 	 基于分层的架构
-    可扩展的自定义DAO层


后续计划：
-    一个通用的自定义线程池
-	 一个高可用的异步的io服务
-	 一个高性能的网络框架-netty
-	 缓存服务--redis
-	 消息中间件服务--kafka
-	 mongodb文件存储数据库
-	 基于数据库的分布式锁
-	 基于redis的分布式锁
-	 应用程序协调服务-zookeeper
-	 分布式事务

应用计划：
-	 基于该框架,构架一个千万级别的数据解析并入库应用
